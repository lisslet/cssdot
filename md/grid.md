# Grid


__Default__:
```html
<ul class="dot-grid-[column][width] _gap[gap][width]">
    <li>1</li>
    <li>2</li>
    <li>3</li>
</ul>
```

__Legacy__:
```html
<ul class="dot-grid-[column][width] _gap[gap][width]">
    <li class="dot-grid">1</li>
    <li class="dot-grid">2</li>
    <li class="dot-grid">3</li>
</ul>
```


## [column]

* `1` ~ `12`
* `x`: display all on 1 row

## Suffixs

### gap

#### [gap]
* 0
* 5
* 10
* 15
* 20
* 25
* 30
