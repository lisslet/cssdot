## Styling Methods

color modification methods on styling map

* `[amount]`    
    transparent color
* `[color]`, `[amount]`      
    mixin color
* `bright`, `[amount]`  
    lighten color
* `invert`  
    invert color

> * [amount]: `-100 ~ 100` the percent
